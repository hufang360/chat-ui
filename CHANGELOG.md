# 版本历史
## v1.1.0 (2026-05-31)

移动端兼容
- 解决看不到顶栏和底栏的情况

对话列表重构
- 文件夹：支持创建文件夹，可将对话折叠至文件夹内，实现分类管理。
- 优化拖拽效果：拖拽时目标位置显示更清晰的横线指示，提升操作直观性。
- 导入 JSONL：支持导入 Claude Code 对话记录 （.jsonl）为一个新对话，实验性！
- 导入 Markdown：能导出，就能导入。
- 清理空对话：清理哪些没有对话记录的空对话。

对话区域
- 补充 行内代码、引用 缺失样式
- 支持脚注功能
- 上标（superscript）和下标（subscript）样式暂未完美支持
- 暂无计划支持数学公式（LaTeX/MathJax）和 Mermaid 图表
- 将用户消息气泡设为绿色。
- 长消息将自动折叠。
- AI回复时，向上滚动浏览历史消息，不会被强制拉回底部。

对话操作菜单
- 复制对话：复制当前对话。
- 重命名对话：修改对话名称。
- 生成对话名：自动使用聊天所选的模型生成话题名称。
- 导出为Markdown：将对话内容导出为Markdown格式文件。格式参考了Cherry Studio。图片不会导出
- 删除：删除当前对话

供应商
- 增加 Gemini 和 xAI
- 适配 Gemini OpenAI 兼容端点，支持 reasoning_effort 思考参数

## v1.0.2 (2026-05-27)

### 修复
- 误删消息：点击用户消息下方的重新生成，会删除已发的对话消息。

### 新功能
- hashtag：通过给url传递参数的方式，实现快速聊天、激活提示词、导入供应商信息。

### 调整
- 编辑用户消息时，新增 `保存并重新生成` 按钮
